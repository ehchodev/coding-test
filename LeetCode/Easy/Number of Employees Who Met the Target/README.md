# 2798. Number of Employees Who Met the Target

**There are `n` employees in a company, numbered from `0` to `n - 1`.**  
어떤 회사에 `0`부터 `n - 1`까지 번호가 붙은 `n`명의 직원이 있다.  

**Each employee `i` has worked for `hours[i]` hours in the company.**  
각 직원 `i`는 회사에서 `hours[i]` 시간동안 일했다.  

**The company requires each employee to work for at least `target` hours.**  
회사는 각 직원이 적어도 `target` 시간 이상은 일하길 요구한다.  

**You are given a 0-indexed array of non-negative integers `hours` of length `n` and a non-negative integer `target`.**  
0부터 시작하는 길이 `n`의 음이 아닌 정수 배열 `hours`와 음이 아닌 정수 `target`이 주어진다.  

**Return the integer denoting the number of employees who worked at least `target` hours.**  
적어도 `target` 시간 이상 일한 직원의 수를 반환하라.
